This is a REST API–based Employee Management System built using Node.js and Express.
It allows managing employee data with create, read, update, and delete operations through a web interface.

Employee records are stored in a JSON file.

Technlogies Used 
Node.js
Express.js
CORS
HTML, CSS, JavaScript
REST API

Structure
employee_backend_node/
├── server.js
├── employees.json
└── public/
    ├── index.html
    ├── style.css
    └── script.js

How to run
# Install required packages
npm install express cors

# Start the server
node server.js

Open in Browser
http://127.0.0.1:3000
